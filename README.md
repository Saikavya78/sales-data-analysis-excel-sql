# Sales Data Analysis using Excel & SQL

## Project Overview
This project analyzes sales data to identify trends, insights, and opportunities for improving business performance. Excel and SQL were used for data cleaning, analysis, and visualization.

## Files
- `sales_data.xlsx` : Excel file containing the dataset, pivot tables, and charts.
- `sales_queries.sql` : SQL queries used to extract insights from the dataset.

## Key Insights
- Identified top-performing products and regions.
- Highlighted seasonal trends in sales.
- Provided actionable recommendations for sales strategy.

## Tools Used
- Microsoft Excel
- SQL (MySQL / PostgreSQL)
